### Local Development
1. Clone the repo: `git clone https://github.com/Lezzur/ruzzel-maestro-portfolio.git`
2. Open `index.html` in your browser
3. Make changes, commit, and push to deploy

## Making Changes

1. Edit files locally
2. Test in your browser (live reload if using a dev server)
3. Commit: `git add . && git commit -m "description"`
4. Push: `git push origin main`
5. Netlify auto-deploys within seconds
